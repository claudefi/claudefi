# Skills

Community and paid trading skills for claudefi.

## What are Skills?

Skills are Claude Code skills that extend Claudefi's capabilities. They follow the [official Claude Code skill format](https://github.com/anthropics/skills/tree/main/skills/skill-creator) with:

```
skill-name/
├── SKILL.md        # Main skill prompt
├── scripts/        # Optional helper scripts
├── references/     # Reference documentation
└── assets/         # Images, data files, etc.
```
